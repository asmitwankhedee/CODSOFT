# 🚢 Titanic Survival Prediction using Machine Learning

## 📌 Overview
This project predicts whether a passenger survived the Titanic disaster using Machine Learning. The model is trained on the Titanic dataset and uses classification techniques to make accurate predictions.

---

## 📊 Key Highlights
- 🔍 Algorithm: Random Forest Classifier  
- 📈 Accuracy: **81%**  
- 🧹 Data Cleaning & Preprocessing  
- 🛠️ Feature Engineering  
- 📉 Model Evaluation using Confusion Matrix  

---

## ⚙️ Technologies Used
- Python  
- Pandas  
- NumPy  
- Scikit-learn  
- Matplotlib / Seaborn  

---

## 🔍 Project Workflow
1. Data Collection  
2. Data Cleaning  
3. Feature Engineering  
4. Model Training (Random Forest)  
5. Model Evaluation  
6. Prediction  

---

## 📊 Model Performance
The model achieved an accuracy of **81%**, indicating good performance on the dataset.
